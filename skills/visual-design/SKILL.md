Create original visual designs through a two-phase process: first articulate a design philosophy, then express it on a canvas as a downloadable artifact.

## When to use

- User asks to create a poster, artwork, design, or visual composition
- User wants a design philosophy or aesthetic direction for a project
- User requests a PDF/PNG with strong visual design quality
- User says "design something," "make a poster," "create a visual," or similar

## Phase 1 -- Design Philosophy

Before touching any canvas, create a visual philosophy (aesthetic movement) that will guide the work.

### Name the movement

Pick 1-2 evocative words: "Brutalist Joy", "Chromatic Silence", "Analog Meditation", "Organic Systems", "Geometric Silence".

### Articulate the philosophy (4-6 paragraphs)

Express how the philosophy manifests through:
- **Space and form** -- mass, void, sculptural qualities
- **Color and material** -- palette logic, chromatic weight
- **Scale and rhythm** -- repetition, contrast, pacing
- **Composition and balance** -- grid vs organic, tension, hierarchy
- **Visual expression** -- how ideas communicate without paragraphs

### Guidelines for the philosophy

- **Emphasize craftsmanship repeatedly** -- the final work must appear meticulously crafted, labored over with care, the product of deep expertise.
- **Leave creative space** -- be specific about direction but concise enough that execution has room for interpretation.
- **No redundancy** -- each design aspect mentioned once unless adding new depth.
- **Minimal text as visual accent** -- text is sparse, essential-only, integrated as a visual element. Never lengthy blocks.

### Aesthetic direction framework

Before committing to the philosophy, consider:
1. **Purpose** -- What is this for? Who encounters it?
2. **Tone** -- Pick an extreme: brutally minimal, maximalist, retro-futuristic, organic, luxury, editorial, brutalist, art deco, industrial, etc.
3. **Differentiation** -- What makes this UNFORGETTABLE?

Save the philosophy as a `.md` file.

## Phase 2 -- Deduce the Subtle Reference

Identify any subtle conceptual thread from the user's request. The topic is a **refined, niche reference embedded within the art** -- not literal, always sophisticated. Someone familiar with the subject should feel it intuitively; others simply experience a masterful composition.

## Phase 3 -- Canvas Creation

Express the philosophy on a canvas using both the design philosophy and the conceptual framework.

### Execution principles

- Create one single page (unless more requested), highly visual, design-forward output as PDF or PNG.
- Use repeating patterns and perfect shapes. Treat the philosophy as a scientific bible -- dense accumulation of marks, layered patterns that build meaning through patient repetition.
- Add sparse, clinical typography and systematic reference markers suggesting a diagram from an imaginary discipline.
- Anchor the piece with simple phrase(s) positioned subtly.
- Limited, intentional, cohesive color palette.
- Embrace the paradox of analytical visual language to express human experience.

### Typography

- Use different fonts for visual variety. Search `./canvas-fonts` directory for available options.
- Most text should use thin fonts. All typography must be design-forward.
- Download fonts as needed to make typography part of the art itself.
- Avoid generic fonts (Inter, Roboto, Arial). Choose distinctive, characterful fonts.

### Spatial composition

- Unexpected layouts. Asymmetry where appropriate. Overlap. Diagonal flow.
- Nothing falls off the page. Nothing overlaps unintentionally.
- Every element contained within canvas boundaries with proper margins.
- 0.5" minimum margins. Breathing room between all elements.

### Color & theme

- Commit to a cohesive aesthetic. One dominant color (60-70% visual weight), 1-2 supporting tones, one sharp accent.
- Dominant colors with sharp accents outperform timid, evenly-distributed palettes.
- Dark/light contrast for visual drama.

### Visual details

- Create atmosphere and depth rather than defaulting to solid colors.
- Contextual effects: gradient meshes, noise textures, geometric patterns, layered transparencies, grain overlays.
- Every detail placed with precision of a master craftsman.

### Quality bar

- The composition, spacing, color choices, typography -- everything must signal expert-level craftsmanship.
- The work should look like it took countless hours by someone at the absolute top of their field.
- Double-check: nothing overlaps, formatting is flawless, every detail perfect.

Output as a downloadable `.pdf` or `.png` alongside the design philosophy `.md`.

## Phase 4 -- Refinement Pass

After the first output, take a second pass to refine. Do NOT add more graphics. Instead:

1. Make the existing composition more cohesive with the art.
2. Respect minimalism -- remove before adding.
3. If the instinct is to draw a new shape, STOP and ask: "How can I make what's already here more of a piece of art?"
4. Crisp edges, perfect alignment, harmonious spacing.

## Multi-page (when requested)

Create additional pages as variations on the same philosophy but distinctly different. Treat the first page as one entry in a coffee table book. Later pages are unique twists -- almost telling a story in a tasteful way. Bundle in the same PDF or as multiple PNGs.

## Gotchas

- **Copyright** -- create original visual designs. Never copy existing artists' work.
- **Font availability** -- always check `./canvas-fonts` before assuming a font exists. Download if needed.
- **Overworking** -- the refinement pass is about precision and cohesion, not adding more. Less is more.
- **Generic aesthetics** -- if the output looks like a template or "AI art," push further. Distinctive choices beat safe ones.
- **Text overload** -- the #1 failure mode. If text exceeds 10% of visual real estate, cut aggressively.
