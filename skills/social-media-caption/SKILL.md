## When to use
- Writing captions for Instagram posts, Reels, TikTok, LinkedIn, YouTube, or any social media
- Trigger: "write a caption", "social media post text", "caption for instagram", "social media text"
- Works for any niche or topic

## Steps

1. **Get the context from the user:**
   - What's the post about? (video topic, product launch, reflection, quote, etc.)
   - Which platform? (Instagram, TikTok, LinkedIn, etc.)
   - Any specific tone? (urgent, reflective, sales, inspirational)
   - If there's an accompanying video script, ask for it or read it from the vault.

2. **Write using the HOOK + PAS + CTA + HASHTAGS framework:**

   **HOOK (1-2 sentences)**
   - Must stop the scroll immediately
   - Use: a bold statement, a contradiction, a statistic, a direct quote, or a provocative question
   - Never start with "Today I want to talk about..." or generic greetings

   **PROBLEM (2-3 sentences)**
   - Name the pain. What's the struggle? What's the lie people believe?
   - Be specific -- not "people suffer" but the exact way they feel it

   **AGITATION (3-5 sentences)**
   - Deepen the wound. Show consequences of inaction
   - Use contrast (Person A vs Person B, before vs after, lie vs truth)
   - If there's a book, product, or resource to reference, introduce it naturally here
   - Make the reader feel the weight before the relief comes

   **SOLUTION (2-3 sentences)**
   - The turn. The answer. The hope.
   - Must be direct and actionable, not vague inspiration
   - Tie back to the hook for closure

   **CTA -- Call to Action (1-2 sentences)**
   - One clear action: comment, save, share, click link, follow
   - Never ask for 3 things -- pick one primary CTA
   - Optional: a question to drive comments

   **HASHTAGS**
   - 8-15 hashtags, mix of broad and niche
   - Include branded hashtag if applicable
   - Platform-specific: Instagram allows 30 but 10-15 is optimal; LinkedIn 3-5; TikTok 5-8

3. **Format the output as copy-paste ready:**
   - No markdown, no bold tags, no labels, no structural markers
   - Plain text with line breaks between sections
   - Hashtags at the bottom, separated by spaces
   - The user should be able to select all -> paste -> publish

4. **Save to vault** at an appropriate path (e.g. `social-captions/{slug}.md`).

## Gotchas
- Caption length: Instagram allows 2,200 chars but the first 2 lines are what matters (the "more" cutoff). Hook must land before that fold.
- LinkedIn tone is more professional -- adjust language but keep the same framework.
- TikTok captions should be shorter -- compress to HOOK + short PAS + CTA + fewer hashtags.
- Always write in the user's preferred language unless specified otherwise.
- If the caption accompanies a video script, read the script first so the caption reinforces -- not repeats -- the video content.
